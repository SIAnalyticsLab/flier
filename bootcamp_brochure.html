Executive Programme Overview
What Is the Accelerated Data Analytics Bootcamp?

The Accelerated Data Analytics Bootcamp is a structured, six-module cohort programme that equips working professionals with practical, deployable skills across three of the most widely used analytical tools in modern organisations: Python, Microsoft Excel, and Microsoft Power BI.

The programme is designed to address a persistent gap between the growing demand for data literacy in organisations and the availability of professionals who can confidently prepare, analyse, and communicate data. Many professionals work with data daily but lack a structured foundation in the tools and methods that would make their work more accurate, efficient, and impactful.

This bootcamp provides that foundation — not through isolated tool training, but through a deliberately integrated curriculum that mirrors how data actually moves through a real analytical workflow: from raw, messy datasets to clean, structured data, through rigorous analysis, and ultimately into clear, decision-ready dashboards and reports.

Python handles the heavy lifting of data preparation and analysis. Excel provides a familiar and powerful environment for structured reporting and business calculations. Power BI translates the work into interactive dashboards that communicate findings to stakeholders at every level of an organisation. Together, these three tools form a coherent, complementary analytical ecosystem.

The bootcamp is delivered using a flipped-classroom model, combining pre-recorded instruction, professionally prepared learning materials, live instructor-led workshops, and applied assignments. Learners progress as a cohort, supported by mentors and a peer learning community, and complete the programme with a portfolio-worthy capstone project.

The Accelerated Data Analytics Bootcamp uses a flipped-classroom model in which learners engage with instructional content before the live session — freeing workshop time for guided practice, problem solving, and applied work. This approach places active learning at the centre of the experience, with instructor time used where it adds the most value: working through real problems alongside learners rather than delivering passive lectures.

Technology Ecosystem
Three Tools, One Integrated Workflow

Each tool in the bootcamp plays a distinct and complementary role. They are not interchangeable — they are sequential stages in the journey from raw data to strategic insight.

Google Colab
Browser-based Python environment requiring no local setup. Provides a consistent, accessible coding space for all learners regardless of operating system.
Python
The analytical engine of the programme. Used for importing, inspecting, cleaning, transforming, grouping, and preparing data for downstream analysis and reporting.
Pandas
Python's primary data manipulation library. Provides Series and DataFrame structures for structured data work, used throughout Modules 1 to 3.
NumPy
Foundational numerical computing library. Supports array operations and underpins Pandas — introduced in Module 2 where relevant to data preparation work.
Microsoft Excel
An indispensable professional tool. Module 4 builds intermediate competency in functions, PivotTables, and structured reporting — skills used alongside and independent of Python.
Power BI Desktop
The primary visualisation and business intelligence platform. Used in Module 5 to build data models, write measures, and develop interactive executive dashboards.
Power Query
Power BI's built-in data transformation layer. Learners use Power Query to import, clean, and reshape data — extending their Python data preparation skills into the BI environment.
DAX
Data Analysis Expressions — Power BI's formula language. Learners write practical DAX measures for calculated KPIs, conditional aggregations, and comparative metrics.

Learning Objectives

  • Filter rows and select columns using loc, iloc, and boolean indexing
  • Sort data and apply conditional transformations
  • Handle missing values: identify, fill, and drop appropriately
  • Clean text data: strip whitespace, standardise case, correct entries
  • Create new variables through calculation and recoding
  • Export clean datasets to CSV and Excel

  • Importing CSV and Excel files with pd.read_csv() and pd.read_excel()

  • Selecting columns, filtering rows, conditional selection
  • Sorting: sort_values(), sort_index()
  • Renaming columns, standardising naming conventions
  • Data type management: astype(), datetime conversion
  • Missing values: isnull(), dropna(), fillna(), interpolation
  • Duplicate records: duplicated(), drop_duplicates()

SI Analytics Frequently Asked Questions
FAQs
Frequently Asked Questions
Is this bootcamp suitable for complete beginners?
Yes. The programme is designed to be fully accessible to learners with no prior coding or analytics experience. Module 1 begins from first principles — Python syntax, the Google Colab environment, and core programming concepts — and builds progressively. No prior knowledge of Python, Excel beyond basic use, or Power BI is required or assumed.
How much time should I expect to commit each week?
Plan for approximately 8 to 12 hours per module week. This includes pre-work (2–3 hours), the live workshop (2 hours), the module assignment (3–4 hours), and any optional help session time. Learners who invest consistently in their pre-work and assignments tend to get the most from the live sessions.
What computer do I need?
You will need a computer with a reliable internet connection. Python is taught using Google Colab, which runs entirely in your web browser and requires no local installation. Power BI Desktop is designed for Microsoft Windows; learners using Windows will have access to the full Power BI experience. If you are on macOS, please contact SI Analytics for guidance before enrolling.
Is attendance at the live workshops compulsory?
Live workshop attendance is strongly encouraged. The workshops are where instructor-guided practice, live problem solving, and direct peer interaction take place — these are difficult to replicate through recordings alone. Optional help sessions are available as an additional resource. Please check with SI Analytics regarding attendance policies for your specific cohort.
What happens if I miss a session?
Learners who miss a live workshop are responsible for reviewing available recordings and completing the module assignment before the deadline. Optional help sessions provide an opportunity to address questions arising from missed content. Consistent non-attendance may affect overall performance. Contact SI Analytics if you anticipate a significant disruption.
Do I need Microsoft Excel already installed?
Microsoft Excel is required for Module 4. A Microsoft 365 subscription or a compatible standalone Excel installation is recommended. Please confirm your access to Excel before the programme begins and contact SI Analytics if you require guidance on software options.
Is Power BI free to use?
Power BI Desktop — which is used throughout Module 5 and the capstone — is available as a free download from Microsoft and is the version used in this programme. Power BI's cloud-based sharing and publishing features may require a Microsoft Power BI Pro licence; these features are covered in the curriculum where available, and SI Analytics will confirm the scope of publishing activities for your cohort.
I already know Excel. Will Module 4 still be useful?
Module 4 is designed as an intermediate-level module, not a basic introduction. If you use Excel regularly but have not worked extensively with structured table referencing, XLOOKUP, INDEX/MATCH, nested conditional logic, or audit-safe PivotTable practice, you are likely to find the module both valuable and challenging. Learners with strong existing Excel skills typically still benefit from the analytical framing and workflow integration that Module 4 provides.
What kind of datasets will we work with?
Modules 1 to 5 use structured datasets provided as part of the programme materials — designed to reflect the kinds of data professionals encounter in public health, business, finance, and development contexts. For the capstone, learners may use a dataset provided by SI Analytics or a suitable publicly available dataset relevant to their professional domain. Learners wishing to use organisational data should ensure they have appropriate approval to do so.
What support is available if I am struggling?
Optional help sessions are available every module for questions, troubleshooting, and additional practice. Instructor feedback on assignments provides structured guidance on areas for improvement. The cohort community also serves as a peer support resource. If you are experiencing significant difficulty, contact your instructor or SI Analytics directly.
Will I receive a certificate?
Learners who successfully meet the programme's completion requirements receive an SI Analytics Certificate of Completion. The recommended minimum threshold is 75% overall (subject to final confirmation). The certificate confirms participation and performance in the Accelerated Data Analytics Bootcamp and is awarded by SI Analytics.
How is the capstone project assessed?
The capstone accounts for 35% of the overall grade and is assessed against a published rubric covering problem definition, data preparation, analytical accuracy, data modelling, dashboard functionality, visual design, insight quality, documentation, and presentation quality. Full rubric details are provided on page 14 of this brochure and will be shared with learners at the start of Module 5.
